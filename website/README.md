# Green AI Tech Website

AI Engineer GitHub profile website built with Next.js, TailwindCSS, and Framer Motion.

## Development

Install dependencies and start the local development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
npm run lint
npm run build
```

## Stack

- Next.js
- TailwindCSS
- Framer Motion
- Lucide React

## Content

Main content is configured in [`src/app/page.tsx`](./src/app/page.tsx).

The hero background image is stored in [`public/images`](./public/images).
